# 🏸 Flappy shuttle 

A beginner Unity (C#) project focused on reflex-based gameplay, where the goal is to avoid a badminton net and survive as long as possible.

## 🎮 Gameplay

* Stay airborne using simple controls
* Avoid colliding with the net
* Survive as long as possible to increase your score

## 🛠 Tech Stack

* Unity
* C#
* VS Code

## 🚀 Running the Project (Source Code)

1. Clone or download this repository
2. Open the project using Unity Hub
3. Launch the project in Unity Editor
4. Click **Play**

## ▶️ Play the Game (Build)

A playable version of the game is available in the **Releases** section.

1. Download the ZIP file from Releases
2. Extract it
3. Run `My project.exe`

## 📂 Project Structure

* `Assets/` – game scripts, sprites, and scenes
* `Packages/` – project dependencies
* `ProjectSettings/` – Unity configuration

## 📌 Notes

* This is a beginner-level project built to explore Unity fundamentals such as physics, collision, and game loops
* The focus is on learning core mechanics rather than full-scale game development
